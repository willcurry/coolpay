## Setup

### Install the gems:

`bundle install`

### Set the required enviroment variables:

`export COOLPAY_APIKEY='xxxxxx'`

`export COOLPAY_USERNAME='xxxxxx'`

### Run the project:

`rackup`

### Run the tests:

`rspec`

## Assumptions:

- UI/UX is not a concern
- The user knows what they are doing
